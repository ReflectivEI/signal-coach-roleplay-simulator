# Rollback Governance Framework

## Activation Criteria
- Any Sev1 governance failure.
- Any protected-path uncertainty.
- Deterministic, ontology, or explainability continuity break.

## Decision Authority
- Governance Council + relevant domain owner(s).

## Rollback Sequence
1. Freeze active pilot/change path.
2. Restore last certified baseline.
3. Re-run deterministic parity.
4. Re-run ontology conformance.
5. Re-run explainability continuity.
6. Re-run manager stability validation.
7. Record constitutional incident and remediation.

## Re-entry Requirements
- Failed gate evidence resolved.
- New certification pack approved.
- Escalation record closed by governance authority.
