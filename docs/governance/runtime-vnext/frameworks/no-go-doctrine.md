# Constitutional NO-GO Doctrine

Default state: NO-GO until certified.

Automatic NO-GO triggers:
- protected-path uncertainty
- deterministic mismatch
- ontology flattening/collapse
- explainability discontinuity
- evidence/rationale disconnect
- manager interpretation instability
- score-first regression
- rollback insufficiency
- missing checkpoint evidence

NO-GO can be lifted only through checkpoint progression with constitutional authority sign-off.
