# Controlled Pilot Governance Constitution

## Pilot Scope
Constrained pilot only. No protected-surface modification.

## Mandatory Controls
- feature flags
- kill switch
- rollback-ready state
- checkpoint evidence lock
- drift telemetry
- protected-path monitoring

## Pilot Entry Requirements
- G0-G6 evidence complete
- deterministic continuity certification
- ontology continuity certification
- explainability continuity certification
- manager stability certification
- QA Twin calibration certification
- score-first regression certification

## Pilot Termination Conditions
- any Sev1 event
- protected-path uncertainty
- explainability break
- deterministic mismatch

## Post-Pilot Requirement
Governance Council review with pass/fail and explicit NO-GO or expand decision.
