## Detection Model
- Static scans: prohibited language and protected-term checks.
- Structural scans: protected-path dependency and continuity checks.
- Semantic audits: ontology and explainability evidence reviews.
- Behavioral parity checks: state-transition and manager-output stability.

## Containment

- Sev1 classes trigger immediate escalation and rollback readiness activation.
